PHP CRUD Form Project

A simple full-stack application demonstrating form handling, data storage, and management using PHP and MySQL.

Project Overview

This PHP CRUD Form Project allows users to submit, view, edit, and delete records through a web interface. It demonstrates the core concepts of server-side programming with PHP, database interactions using MySQL, and responsive frontend design with Bootstrap.

  • Frontend: HTML, CSS, Bootstrap
  • Backend: PHP
  • Database: MySQL
  • Features: Add, View, Edit, Delete Records, Form Validation
