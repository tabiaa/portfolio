Book Reading App

A cross-platform mobile app built using Flutter and Dart.

Project Overview

The Book Reading App is a mobile application that allows users to easily explore and read books. Each book includes a short summary for quick reference, and users can open or download the PDF to read the full content. The app provides a clean and distraction-free reading experience.

  • Framework: Flutter
  • Language: Dart
  • Platform: Android & iOS
  • Features: Browse Books, View Summaries, Read PDFs, Download PDFs
