Complaint Management System (CMS)

Full-stack project built using React, Node.js, Express, and MySQL.

Project Overview

The Complaint Management System (CMS) is a full-stack application designed to streamline complaint handling. Supervisors can assign tasks to fitters, track complaint status, and generate reports. Fitters can update task statuses in real-time, ensuring transparency and efficiency.

  • Frontend: React.js with Bootstrap
  • Backend: Node.js, Express.js
  • Database: MySQL with normalized schema
  • Features: Authentication, Role-based Access, Dynamic Dashboard, Task Assignment
